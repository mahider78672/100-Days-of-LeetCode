✅ Day 50 – Problem: Minimum Operations to Convert All Elements to Zero

Problem:
In one operation, subtract the smallest non-zero value from all non-zero elements.
Return how many such operations are needed.

Approach:
• Observed mathematical pattern:
    Answer = Count of unique positive numbers in the array.
• Used a Set to track distinct positive values.

Complexity:
Time: O(N)
Space: O(N)

Language: Java ☕
