✅ Day 44 – Problem: Reverse Nodes in k-Group (LeetCode 25)

Problem:
Reverse nodes of a linked list in groups of size k. If remaining nodes < k, leave unchanged.

Approach:
1. Check if k nodes exist ahead.
2. Reverse the k nodes using pointer manipulation.
3. Recursively process the rest and attach.

Complexity:
Time: O(n)
Space: O(1)

Language: Java ☕
